# 安装前准备

## 卸载杀毒软件

如果您使用的是Win10系统，那么我建议您卸载电脑中全部的杀毒软件。Win10系统自带的安全工具已经足够日常保护您电脑的安全。

杀毒软件包括且不仅限于：360系（包括安全卫士、杀毒和浏览器等一系列软件）、Mcafee、金山毒霸等。
{: .notice--warning}
